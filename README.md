👟 Nike Sales Dashboard

This project presents a comprehensive Sales Dashboard for Nike, designed to analyze and visualize sales performance across different dimensions such as product categories, regions, gender segments, and yearly trends.

The dashboard helps in understanding business performance and supports data-driven decision-making through interactive and visually appealing charts.

🚀 Project Overview

The dashboard includes:

📊 Total Units Sold: 1,695
💰 Total Revenue: 521.41K
📈 Profit & Sales trends over years
🧑‍🤝‍🧑 Customer segmentation by gender
🌍 Regional sales distribution
👟 Product-wise performance analysis
